关于PC和M站标签值验证说明
一.首页
1.pc站
<meta name="applicable-device" content="pc">
<meta name="mobile-agent"content="format=html5; url=https://m.tianyancha.com/">
<link rel="alternate" media="onlyscreenand(max-width:640px)" href="https://m.tianyancha.com/" />
2.m站
<meta name="applicable-device" content="mobile">
<link rel="canonical" href="https://www.tianyancha.com/">


二.其他页面 如果url 和 href 对应的是 页面的url。
1.m站
<meta name="applicable-device" content="mobile">
<link rel="canonical" href="https://www.tianyancha.com/company/22822">
2.pc站
<meta name="applicable-device" content="pc"><meta name="mobile-agent"content="format=html5; url=https://m.tianyancha.com/company/22822"><link rel="alternate" media="onlyscreenand(max-width:640px)" href="https://m.tianyancha.com/company/22822" />
